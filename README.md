# 🔮 今日运势测算 H5（含分享海报生成）

一个精美的 H5 今日运势测算页面，支持生成分享海报。  
适配移动端，UI 采用玻璃拟态 + 渐变星空背景，结果展示带动画进度条，并可一键生成可保存的长图海报（Canvas 绘制）。

👉 在线演示：  
**https://guiyan201213-dev.github.io/fortune-h5/**

---

## ✨ 功能特性

- ✅ **输入信息测运势**：支持姓名（可选）、星座选择、日期选择  
- ✅ **稳定随机**：同一输入在同一天会生成一致结果（seed 随机）  
- ✅ **多维度运势评分**：总评 / 事业 / 财运 / 感情 / 健康（进度条动画）  
- ✅ **幸运提示**：幸运色 / 幸运数字 / 幸运时段 / 幸运物  
- ✅ **生成分享海报**：一键生成精美海报，手机长按保存、电脑可下载  
- ✅ **移动端友好**：响应式布局，适合公众号 H5 / WebView / 活动页  
- ✅ **纯前端**：无需后端，无第三方依赖，开箱即用

## 🚀 快速开始（本地运行）

这是一个**单文件静态网页**，无需安装依赖：

1. 下载或复制本仓库代码  
2. 双击打开 `index.html`（或用手机浏览器打开）

## 🧩 技术实现说明

- **UI**：纯 CSS 实现玻璃拟态卡片、渐变背景、星空漂浮动画
- **逻辑**：JS 使用 seed random（mulberry32 + hashSeed）生成稳定结果
- **海报**：Canvas 生成 1080×1920 海报，可保存/下载
- **兼容性**：主流 Android / iOS 浏览器均可运行

## ❓常见问题（FAQ）

### 1）海报为什么无法直接保存？
- ✅ 手机端：建议长按海报图片保存到相册  
- ✅ 电脑端：点击「下载海报」按钮即可  
某些 WebView / 微信浏览器可能会限制自动下载，这属于浏览器行为。

### 2）想改成塔罗牌 / 抽签 / 生肖 / 八字怎么办？
本项目结构清晰，可在 `genFortune()` 中替换运势生成规则；  
也可以在海报绘制 `drawPoster()` 中替换文案与布局。

### 3）如何自定义配色和文案？
- UI 配色：修改 `:root` 中的变量或背景渐变即可  
- 运势文案：修改 `actions` / `overviewText` 数组即可
